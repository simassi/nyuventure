# 🌍 NyuVenture

NyuVenture adalah platform digital yang membantu pengguna menemukan destinasi wisata, kuliner, penginapan, dan transportasi dalam satu tempat yang ringkas, praktis, dan mudah digunakan.

## ✨ Fitur Utama
- Destinasi Terkurasi
- Dekat Anda
- Penginapan Nyaman
- Transportasi Praktis
- Kuliner Lokal
- Tiket Online

## 🚀 Cara Menjalankan
```bash
npm install
npm run dev
